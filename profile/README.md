# 中国科学技术大学乒乓球协会

本组织用于存放中国科学技术大学乒乓球协会学生维护的软件项目、技术文档和相关协作工具。

> 本组织由协会学生成员维护，不代表中国科学技术大学官方 GitHub 组织。

## 项目

* [`competition-platform`](https://github.com/USTC-Student-TableTennisAssociation/competition-platform)：协会比赛报名、比赛管理、个人主页、积分与排行榜系统。该仓库主要面向组织成员开放。

## 维护者

本组织由负责技术开发、系统运维和项目传承的现任及往届协会成员维护。

## 说明

本组织中的仓库主要用于协会内部的信息化建设，包括比赛系统、数据管理、协作工具和相关文档。
